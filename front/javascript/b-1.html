# 按功能划分

# 赋值操作符

操作符 描述 例子 等同于 结果
= 直接赋值 var num = 10 var num = 10 10
+= 相加赋值 var num += 10 var num = num + 10 20
-= 相减赋值 var num -= 10 var num = num - 10 0
*= 相乘赋值 var num *= 10 var num = num * 10 100
\= 相除赋值 var num \= 10 var num = num \ 10 1
%= 求余赋值 var num %= 10 var num = num % 10 0

# 算术操作符

操作符 描述 例子 结果
+ 直接赋值 var num = 10 + 10 20
- 相加赋值 var num = 20 - 10 10
* 相减赋值 var num = 10 * 10 100
\ 相乘赋值 var num = 100 \ 10 10
% 相除赋值 var num = 10 % 3 1

# 比较操作符

操作符 描述 例子 结果
> 大于 3 > 4 false
< 小于 3 < 4 true
>= 大于等于 3 >= 4 false
<= 小于等于 3 <= 4 true
== 等等 3 == '3' true
=== 全等 3 === '3' false
!= 不等 3 != '3' false
!== 不等等 3 !== '3' true

操作符 描述 例子 结果
&& 逻辑与 3 && 4 4
|| 逻辑或 3 | 4 3
! 逻辑非 !3 false

# 位移运算符

操作符 描述 例子 等同于 结果
& 位移与 6 & 8 0110 & 1000 0100
| 位移或 6 8 0110
~ 位移非 ~8 ~1000 0111
<< 左移 6 << 2 0110 << 2 011000
>> 右移 6 >> 2 0110 >> 2 0001
>>> 零填充右移 6 >>> 1 0110 >>> 1 0011
&= 位移与赋值 var num = 6; num &= 2; var num = 0110 & 0010 0010
| = 位移或赋值 var num = 6; num | = 2; var num = 0110 | 0010 0110
>>= 右移赋值 var num = 6; num >>= 2; var num = 0110 >> 2 0001
<<= 左移赋值 var num = 6; num <<= 2; var num = 0110 << 2 011000
^ 位移异或 6 ^ 8 0110 ^ 1000 1110

[^注解1]：下面以 5 为例：
二进制原码：0000 0000 0000 0000 0000 0000 0000 0101
取反操作后：1111 1111 1111 1111 1111 1111 1111 1010
而在 Java 中，有符号百整数都是用补码来表示，而补码=反码+1
1.先求度反码：1000 0000 0000 0000 0000 0000 0000 0101
2.再求补码：1000 0000 0000 0000 0000 0000 0000 0110
最高位代表符号位 1 表示负数，0 表示正数 所以结果是-6

[^注解2]：JavaScript 进制转换方法，parseInt(转换的数字,进制).toString(进制)。例如，parseInt(100,10).toString(2)，把十进制的数字 100 转换为二进制数

# 按条目划分

# 一元操作符

概念：只能处理一个操作数的操作符
操作符：++、--、+、-、！、typeof、new ...

# 自增、自减运算符

++、-- 必须配合变量使用
变量在前，先取后加
var i = 1;
console.log(i++,i); // 1 2
变量在后，先加后取
var ii = 1;
console.log(++i,i); // 2 2
配合变量
var iii = 1;
console.log(++(++i)); // Uncaught SyntaxError: Invalid left-hand side expression in prefix operation

# 二元操作符

概念：处理两个操作数的操作符
操作符：&&、||、+、-、*、//、% ...

console.log(1 && window && console.log && {} && false && "abc"); // false
console.log(null || undefined || NaN || 0 || -0 || false || true || ""); // true

[^注解]： 逻辑与或都属于短路运算符
逻辑与&&是找假值的操作符，在未找到假值时，则会一直运算，直到找到假值，则运算停止
逻辑或||是找真值的操作符，在未找到真值时，则会一直运算，直到找到真值，则运算停止

# 三元操作符

概念：处理三个操作数的操作符
操作符：条件 ? 结果1 : 结果2

var result = true ? 1 : 2;
console.log(result); // 1

var score = 90;
var level = score >= 60 ? "及格" : "不及格";
console.log(level); // 及格
