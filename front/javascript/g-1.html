# DOM 描述

  • 当浏览器加载 HTML 页面的时候，首先是 DOM 解构的结算，计算出的 DOM 结构就是 DOM 树（把页面中的 HTML 标签像树状结构一样，分析计算出的层级关系）

# DOM 节点

节点类型 说明 节点名称
（nodeName）		 节点值
（nodeValue）		 节点类型对应值
（nodeType）
元素节点 每一个HTML标签都是一个元素节点，如<div>、<p>、<ul> 大写的标签名称 null 1
属性节点 元素节点（HTML标签）的属性，如id、class、name等 属性名称 属性值 2
文本节点 元素节点或属性节点中的文本内容 #text 文本内容 3
注释节点 表示文档注释，形式为<!-- comment text --> #comment 注释内容 8
文档节点 表示整个文档（DOM TREE 的根节点，即document） #document null 9
<div id="block">html块元素</div>
<!-- 一段注释内容 -->
<script>
    // 元素节点
    var div = document.getElementById("block");
    console.log(div.nodeName, div.nodeValue, div.nodeType);
    // 属性节点
    var divAttr = div.getAttributeNode("id");
    console.log(divAttr.nodeName, divAttr.nodeValue, divAttr.nodeType);
    // 文本节点
    var divText = div.childNodes[0];
    console.log(divText.nodeName, divText.nodeValue, divText.nodeType);
    // 注释节点
    var commentNode = div.nextSibling;
    console.log(commentNode.nodeName, commentNode.nodeValue, commentNode.nodeType);
    // 文档节点
    console.log(document.nodeName, document.nodeValue, document.nodeType);
</script>

# DOM 获取元素使用样式最终值

  • 元素使用的样式，有行内样式、外部引用样式、标签样式，如果不存在样式提升（!important）的情况下，行内样式的优先级最高。获取元素的行内样式，通过元素.style。如果元素使用的是标签样式或者外部引用，则通过以下方式获取：
小脚丫
<style>
#box{
   width: 100%;
   height: 120px;
   margin: 0 auto;
   padding: 40px 0px;
   display: flex;
   justify-content: center;
   align-items: center;
}
#innerBox{
   width: 100px;
   height: 100px;
   text-align: center;
   line-height: 100px;
   color: #fff;
   border: 1px solid #ccc;
   background-color: green;
   animation:theanimation 8s infinite alternate; 
   -webkit-animation:theanimation 8s infinite alternate ; 
   -moz-animation:theanimation 8s infinite alternate ; 
   -o-animation:theanimation 8s infinite alternate ; 
   -ms-animation:theanimation 8s infinite alternate ;
}
@keyframes theanimation{
   0%{transform: perspective(120px) rotate3d(0, 0, 0,0deg);}
   25%{transform: perspective(120px) rotate3d(0.5, 0.5, 0.5,180deg);}
   50%{transform: perspective(120px) rotate3d(1, 1, 1,360deg);}
   25%{transform: perspective(120px) rotate3d(0.5, 0.5, 0.5,180deg);}
   100%{transform: perspective(120px) rotate3d(0, 0, 0,0deg);}
}
</style>
<div id="box" style="background-color: aqua">
  <div id="innerBox">小脚丫</div>
</div>
<script>
    var box = document.getElementById('box')
    ,   innerBox = document.getElementById('innerBox');
    // 当前box盒子使用的背景颜色是行内样式，可以通过以下方式获取：
    console.log(box.style.backgroundColor); // aqua

    // 通过行内样式是获取不到innerBox的背景颜色，原因是innerBox使用的最终样式是标签样式
    console.log(innerBox.style.backgroundColor); // ""

    // 既然行内获取不到，则可以通过以下方式获取：
    console.log(innerBox.currentStyle.backgroundColor); // IE浏览器独有获取方式
    console.log(window.getComputedStyle(innerBox,null).backgroundColor); // IE8以上浏览器和其它主流浏览器都可以通过此方式获取

    // 封装获取样式使用最终值兼容函数
    function getStyle(ele, cssProp){
        return ele.currentStyle ? ele.currentStyle[cssProp] : window.getComputedStyle(ele,null)[cssProp];
    }
</script>

# DOM 表格操作

名称 描述
createCaption() 创建<caption>元素，并返回其引用
deleteCaption() 删除<caption>元素
caption 获取<caption>元素的引用
createTHead() 创建<thead>元素，并返回其引用
deleteTHead() 删除<thead>元素
tHead 获取<thead>元素的引用
createTBody() 创建<tbody>元素，并返回其引用
tBodies 获取所有的<tbody>元素
createTFoot() 创建<tfoot>元素，并返回其引用
deleteTFoot() 删除<tfoot>元素
tFoot 获取<tfoot>元素的引用
insertRow(pos) 向rows集合中的指定位置插入一行并返回其引用
deleteRow(pos) 删除指定的行
rows 获取指定父节点下的所有的行集合
rowIndex 行标
rowSpan 合并行
insertCell(pos) 向cells集合中的指定位置插入一列并返回其引用
deleteCell(pos) 删除指定的列
cells 获取指定行下的所有的列集合
cellIndex 列标
colSpan 合并列
