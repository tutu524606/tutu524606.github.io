    # 变量

    JavaScript 存放数据值的可变容器，使用关键字 var 声明并使用

    var num = 10;
console.log(num); // 10
num = 20;
console.log(num); // 20

    [^注解]: 在上述示例中，var 是声明变量的关键字，num 为量的标识符，=为赋值操作符

    # 标识符

    在 JavaScript 中，变量名称、参数名称、函数名称都是标识符 标识符的命名规则：

    • 必须以字母开头
    • 中间不能包含除$和_之外的特殊字符
    • 区分大小写
    • 不能使用关键字作为标识符，保留字不建议作为标识符使用

    # 关键字和保留字

    关键字

    break case catch continue default delete do else finally for function if in instanceof new return switch this throw try typeof var void while with

    保留字

    abstract boolean byte char class const debugger double enum export extends final float goto implements import int interface long native package private protected public short static super synchronized throws transient volatile

    # 一次声明多个变量并赋值，使用逗号隔开

    var name = "张三",
  age = 18,
  sex = true,
  girlFriend = undefined,
  son = null,
  school;
console.log(name, age, sex, girlFriend, son, school);

    [^注解]：变量声明未赋值，变量的值为 undefined

    # 变量声明提升

    console.log(num); // undefined
var num = 10;
console.log(num); // 10
console.log(num1); // Uncaught ReferenceError: num1 is not defined

    [^注解]：JavaScript 属于单线程执行，程序执行自上而下，控制台第一次输出 num 时，还未声明赋值，但 var 声明存在提升机制，故输出的是 undefined。变量未声明直接使用，则报语法错误

    # var 声明的量，允许重复声明

    var num = 10;
num = 20;
console.log(num); // 20

    # var 允许先声明，后使用

    var num;
num = 20;
console.log(num); // 20

    # var 声明的量只存在函数作用域，不存在块级作用域

    函数作用域内，var声明的量属于函数内私有变量
function fun(){
    var num = 10;
}
console.log(num); // Uncaught ReferenceError: num1 is not defined
块级作用域内，函数属于代码块所在的执行上下文内的全局变量，且存在声明提升
console.log(num); // undefined
{
    var num = 10;
}
console.log(num); // 10

    # 全局环境内 var 声明的变量，会挂载到 window 全局对象上

    var str = "字符串";
console.log(window.str); // 字符串
